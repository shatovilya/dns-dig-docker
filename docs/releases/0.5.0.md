# Release 0.5.0 — Web UI localization (EN/RU)

**Date:** 2026-07-10  
**App version:** `0.5.0` (`app/main.py`)

## Summary

Git tag **v0.5.0** is a cumulative release since v0.3.0: it also ships [0.4.0](0.4.0.md) (PostgreSQL historical persistence), which was not published as a separate GitHub release.

Adds production-ready **English and Russian** localization to the optional Web UI without changing core API contracts, security model, or Prometheus metrics.

## Localization / Russian language support

| Item | Detail |
|------|--------|
| Languages | `en` (default), `ru` |
| Switcher | Header `EN \| RU` — no full page reload |
| Bundles | `app/ui/static/i18n/en.json`, `ru.json` |
| Runtime | `app/ui/static/js/i18n.js` (`DnsDebugI18n`) |
| Persistence | `localStorage` key `dns-debug-lang` (when enabled) |
| Formatting | `Intl` number/percent for locale |

### Env vars

| Variable | Default |
|----------|---------|
| `DNS_DEBUG_UI_I18N_ENABLED` | `true` |
| `DNS_DEBUG_UI_DEFAULT_LANG` | `en` |
| `DNS_DEBUG_UI_SUPPORTED_LANGS` | `en,ru` |
| `DNS_DEBUG_UI_LOCALE_STORAGE_ENABLED` | `true` |

### Scope localized

- All dashboard panels (Live, Historical, Compare)
- Filters, KPIs, charts, tables, modals, banners
- Empty/loading/error/stale/retention states
- `global_status` signals (client-side from `code` + `params`)

### Not translated

- FQDN, resolver hostnames, `system` / `absolute_fqdn`, record types, metric names, raw verdict codes in data

## UX / responsive

- CSS: KPI label wrapping, filter label `white-space`, compact language switcher
- QA: verify EN and RU at 1920, 1440, 1366, 1280, 1024, 768, 390, 375px

## API compatibility

- **Additive only:** `global_status.signals[].params` for i18n interpolation
- English `message` / `disclaimer` / `note` fields preserved as fallback

## Docs / process

- `AGENT.md`, `CLAUDE.md`, `CURSOR.md`, rules, skills, `debugging-checklist.md` updated with i18n agent rules
- `tests/test_ui_i18n.py` — locale key parity, config parsing, signal params

## Upgrade notes

No breaking changes. Set `DNS_DEBUG_UI_I18N_ENABLED=false` for English-only (pre-0.5.0 behavior).
